# WristHub - Premium Watch Store

A modern e-commerce platform for luxury watches built with React, TypeScript, Vite, Tailwind CSS, and Firebase.

## Quick Start

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```

## Documentation

- **[README.md](./README.md)** - Main documentation

## Features

- 🛒 Full e-commerce functionality
- 🔐 Hidden admin panel
- 📱 Mobile-responsive design
- ⭐ Customer review system
- 📦 Order tracking by phone number
- 🎨 Premium dark theme with glassmorphism

## Tech Stack

- React 18 + TypeScript
- Vite
- Tailwind CSS
- Firebase (Firestore + Auth)
- ImgBB (Image hosting)

## Environment Variables

Copy `.env.example` to `.env.local` and fill in your values. See [DEPLOY.md](./DEPLOY.md) for details.
